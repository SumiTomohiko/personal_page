corgi

corgi is a regular expression engine based on Python's one. Yog is using this library. The documentation is at GitHub. Tarballs are available at @repository@.

fawm

fawm is a small window manager. The documentation of fawm is in GitHub. Source code is available at @repository@.

rush

An Audio Player

An Audio Player is an audio player for Android. README is available at GitHub. You can install this software from Google play. Tarballs of source code (AnAudioPlayer-x.y.z.tar.xz) are in @repository@. GitHub is hosting the git repository.
